# MovieAngular

This project uses:
- [Angular CLI](https://github.com/angular/angular-cli) version 6.2.4
- Angular 6
- The Movie Database (TMDB)

## Features
  - Movies category
      - All Movies
      - Genre
      - Movie
  - TV shows category
      - All TV Shows
      - Genre
      - TV Show
  - User
      - Profile
      - Setting
